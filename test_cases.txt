 №1 Поиск билетов туда и обратно (Вильнюс-Амстердам) на 2-х взрослых эконом
	1. Зайти на сайт https://www.aviasales.ru
	2. Ввести город вылета (Вильнюс)
	3. Ввести город прибытия (Амстердам)
	4. Выбрать дату вылета (28 апреля)
	5. Выбрать дату возвращения (1 мая)
	6. Выбрать количество пассажиров (2 взрослых)
		6.1. Нажать на выпадающий список
		6.2. Напротив надписи "Взрослые" нажать кнопку "+" для увеличения количества взрослых с 1 до 2-х
	7. Нажать кнопку "Найти билеты"
Ожидаемый результат: Список подходщих рейсов

 №3 Поиск билетов по "сложному маршруту" (Вилбнюс-Рим-Амстердам-Вильнюс) для 1-го взрослого
	1. Зайти на сайт https://www.aviasales.ru
	2. Выбрать ссылку "Составить сложный маршрут"
	3. Заполнить первую строку
		3.1. Ввести город вылета (Вильнюс)
		3.2. Ввести город прибытия (Берлин)
		3.3. Выбрать дату вылета (28 апреля)
	4. Заполнить вторую строку
		4.1. Ввести город вылета (Берлин)
		4.2. Ввести город прибытия (Амстердам)
		4.3. Выбрать дату вылета (29 апреля)
	5. Добавить третью строку нажав на кнопку "Добавить перелет"
	6. Заполнить третью строку
		6.1. Ввести город вылета Амстердам
		6.2. Ввести город прибытия (Вильнюс)
		6.3. Выбрать дату вылета (1 мая)
	7. Нажать кнопку "Найти билеты"
